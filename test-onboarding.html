Test du tunnel d'onboarding

Pour tester le tunnel :

  1. Allez sur http://localhost:3000
  2. Cliquez sur l'onglet "Créer mon compte"
  3. Remplissez les champs (prénom, nom, email, téléphone, mot de passe)
  4. Cliquez sur le bouton "Créer mon compte"
  5. Le tunnel d'onboarding devrait s'ouvrir
